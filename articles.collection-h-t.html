Сбор по Хэштегам

Функция сбора по хэштегам, служит для сбора аудитории по определенным хэштегам.
Если пользователь хоть раз, к примеру оставлял пометку с хэштегом "здоровье" то соответственно данный пользователь окажется в базе после сбора по хэштегу "здоровье" + все другие пользователи, которые так же использовали хоть раз хэштег "здоровье".

1) Перейдите в раздел "Сбор аудитории"
2) Нажмите на "Указать задачу"

Основным вариантом для сбора аудитории по хэштегам, рекомендуем использовать функцию Boost. Прочитать подробнее о ней можно по ссылке:
http://support.tooligram.com/topics/11571-novaya-funktsiya-boost/
Для функции Boost аккаунты не нужны. Скорость работы Сборов в разы быстрее!
Переключите флаг "Использовать" в активное положение и вставьте в поле "Ключ:" ваш ключ ускорителя.
После, нажмите на "По хэштегам"
Как настроить сам сбор по хеш - тегам смотри далее с пункта 5.

Альтернативный способ сбора с использованием аккаунта.

1) Нажимаете на вкладку "Сбор аудитории" в программе.
2) Нажимаете "Указать задачу"

3) В левое окошко вводите не нужный аккаунт инстаграм в формате логин:пароль, который будет участвовать в сборе.
(Не стоит вводит тот аккаунт, который вы будете продвигать. Лучше использовать не нужный аккаунт или аккаунты. Если вы не знаете, где приобрести аккаунты, то в качестве примера, есть пару сервисов:
http://zismo.biz/forum/70-kupit-i-prodat-akkaunty/
http://djekxa.ru/ )
4) Сверху указываете имя сбора, которое вам нужно(опционально).
5) Выбираете "По хэштегам"

Сбор аудитории по хэштегам.

6) Вводите хэштеги, которые вам нужны, БЕЗ РЕШЕТОК и на любом языке (русский, английский) маленькими буквами, в столбик!
7) Выбираете пустой .txt(Текстовый файл), куда сохранится аудитория.
8) Вписываете то количество, которое вам нужно собрать С КАЖДОГО хэштега
9) Кнопка "Зацикливать" необходима для большего количества сбора по хэштегу, то есть, если количество записей по хэштегу может превышать 500.000, то необходимо активировать "Зацикливание".
10) Нажимаете "Добавить"

11) После нажатия кнопки "Добавить" выделяем задание левой кнопкой мыши и нажимаем "Старт", задание должно перейти в статус "Активно".
